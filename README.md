# Student Exam Performance Analysis 📚

## Overview
Analyzed 1,000+ student records to identify key factors influencing 
Math, Reading, and Writing scores.

## Key Results
- Test preparation raised scores by 10–15%
- Strong correlation found between weekly study hours and performance
- Improved data consistency by 100% through cleaning & transformation
- Created 10+ visualizations across gender, parental education & study habits

## Tech Stack
Python | Pandas | NumPy | Matplotlib | Seaborn | Jupyter Notebook

## Files
| File | Description |
|------|-------------|
| `student_exam_analysis.ipynb` | Main analysis notebook |
| `Student_Exam_Score dataset.csv` | Student performance dataset (Kaggle) |
| `visualizations/` | Folder with all chart screenshots |

## Analysis Steps
1. Data Loading & Exploration
2. Data Cleaning & Transformation (Pandas, NumPy)
3. Exploratory Data Analysis (EDA)
4. Visualization by gender, parental education & study habits
5. Key Findings & Insights

## Sample Insights
- 📖 Students with test prep scored 10–15% higher on average
- 👩‍👨 Parental education level positively correlated with student scores
- ⏱️ Weekly study hours showed strong positive correlation with all 3 subjects
